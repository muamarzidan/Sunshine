HTML, CSS, dan Javascript

Silabus Pelatihan

  • Pengenalan Web Development : Pengenalan tentang web browser, HTML, CSS, dan JavaScript dalam web development; Lingkungan pengembangan (text editor, browser, dll.).
  • Pengenalan HTML : Struktur dasar dokumen HTML; Pengenalan dan Penggunaan tag HTML (heading, paragraf, gambar, footer, dll.); Memahami atribut HTML; Memahami semantic HTML; Latihan HTML.
  • Pengenalan CSS : Pengenalan dan penggunaan CSS; Dasar penggunaan untuk CSS ke dalam dokumen HTML; Selektor dan deklarasi property; Penggunaan selector dan proper CSS; Latihan CSS.
  • System Layout dengan CSS : Box model dalam CSS; Positioning dan display; Flexbox untuk tata letak; Grid layout; Latihan penggabungan HTML dan CSS.
  • Responsif Website : Prinsip desain responsif; Media queries; Menyusun tata letak responsif; Pengoptimalan gambar untuk web.
  • Dasar JavaScript : Pengenalan JavaScript; Variabel, tipe data, dan operasi dasar dll; Pernyataan kondisional dan perulangan.
  • Manipulasi BOM dan DOM dengan JavaScript : Apa itu DOM (Document Object Model); Memilih elemen DOM; Menambah, menghapus, dan mengubah elemen; Event handling; Latihan Javascript.
  • Pengenalan Bootstrap dan React JS
  • Proyek Akhir dan Deployment : Menggabungkan HTML, CSS, dan JavaScript untuk membuat proyek web sederhana; Uji coba proyek dan debugging; Menyusun proyek untuk deployment; Membahas langkah-langkah untuk mempublikasikan proyek ke web.
